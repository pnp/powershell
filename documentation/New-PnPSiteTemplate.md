# New-PnPSiteTemplate

## SYNOPSIS
Creates a new provisioning site template object

## SYNTAX

```powershell
New-PnPSiteTemplate 
```

## DESCRIPTION

Allows to create a new provisioning site template.

## EXAMPLES

### EXAMPLE 1
```powershell
$template = New-PnPSiteTemplate
```

Creates a new instance of a site template object.
