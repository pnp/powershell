# Remove-PnPGroup

## SYNOPSIS
Removes a group from a web.

## SYNTAX

```
Remove-PnPGroup [[-Identity] <GroupPipeBind>] [-Force] [-Web <WebPipeBind>] [-Connection <PnPConnection>]
 [<CommonParameters>]
```

## DESCRIPTION

## EXAMPLES

### EXAMPLE 1
```powershell
Remove-PnPGroup -Identity "My Users"
```

Removes the group "My Users"
