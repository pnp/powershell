# Remove-PnPView

## SYNOPSIS
Deletes a view from a list

## SYNTAX

```
Remove-PnPView [-Identity] <ViewPipeBind> [-List] <ListPipeBind> [-Force] [-Web <WebPipeBind>]
 [-Connection <PnPConnection>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION

## EXAMPLES

### EXAMPLE 1
```powershell
Remove-PnPView -List "Demo List" -Identity "All Items"
```

Removes the view with title "All Items" from the "Demo List" list.
