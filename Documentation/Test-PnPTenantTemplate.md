# Test-PnPTenantTemplate

## SYNOPSIS
Tests a tenant template for invalid references

## SYNTAX

```
Test-PnPTenantTemplate -Template <ProvisioningHierarchy> [-Connection <PnPConnection>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION

## EXAMPLES

### EXAMPLE 1
```powershell
Test-PnPTenantTemplate -Template $myTemplate
```

Checks for valid template references
